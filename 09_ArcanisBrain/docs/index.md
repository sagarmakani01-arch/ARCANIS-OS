# ArcanisBrain Documentation

Welcome to the ArcanisBrain documentation. ArcanisBrain is the central intelligence layer for the Arcanis ecosystem, providing modular AI capabilities including reasoning, memory, multi-agent orchestration, personality adaptation, and security.

## Quick Links

- [Architecture Overview](architecture.md)
- [Modules](modules/reasoning.md)
- [API Reference](api/reference.md)

## Structure

| Module | Description |
|--------|-------------|
| **Reasoning** | Task planning, decision making, problem solving |
| **Memory** | Short/long-term memory, preferences, knowledge retrieval |
| **Agents** | Agent communication, task delegation, tool system |
| **Personality** | Context awareness, communication style, user adaptation |
| **Security** | Permissions, safe execution, audit logging |
| **API** | REST, GraphQL, and WebSocket interfaces |
