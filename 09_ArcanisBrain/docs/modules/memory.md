# Memory Module

Four-tier memory system for context preservation and knowledge retention.

## ShortTermMemory

Episodic buffer with capacity limits and TTL-based expiry. Fast recall via keyword scoring.

## LongTermMemory

Persistent storage backed by JSON files. Supports search, importance scoring, and access tracking.

## UserPreferences

Per-user preference profiles with learning from interaction patterns (sentiment, topic interest).

## KnowledgeBase

Structured knowledge storage with tag-based organization and semantic query support.
