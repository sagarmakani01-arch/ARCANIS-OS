Welcome to the ArcanisLabs documentation framework. This repository is the single source of truth for the Arcanis ecosystem — containing the philosophy, architecture, standards, roadmaps, and research that guide every project in the ecosystem.

## Purpose

ArcanisLabs builds toward an **AI-native operating system** — a ground-up reimagining of computing where artificial intelligence is not an application layer but the core substrate of the system. This documentation defines the vision, architecture, and engineering discipline required to achieve that goal.

## Document Map

| Section | Path | Description |
|---|---|---|
| **Vision** | `vision/` | Philosophy, long-term goals, OS concept, ecosystem principles |
| **Architecture** | `architecture/` | Ecosystem overview, communication, dependencies, security |
| **Standards** | `standards/` | Coding guidelines, conventions, repo structure, testing, docs |
| **Roadmaps** | `roadmaps/` | Short/long-term milestones, project dependency graph |
| **Research** | `research/` | Notes on OS, languages, AI, compilers, hardware |

## How to Use This Documentation

- **New contributors** — Start with `vision/philosophy.md` and `vision/ecosystem-principles.md`
- **Architects** — Read `architecture/ecosystem-overview.md` then `architecture/dependencies.md`
- **Developers** — Follow `standards/` documents for all implementation work
- **Planners** — Consult `roadmaps/` for milestone tracking
- **Researchers** — Explore `research/` for deep technical investigations
