---

This project uses [Semantic Versioning 2.0.0](https://semver.org/).

Given a version number **MAJOR.MINOR.PATCH**:

- **MAJOR** — Incompatible documentation restructuring or foundational shifts in vision
- **MINOR** — New documents or sections added in a backward-compatible manner
- **PATCH** — Corrections, clarifications, formatting improvements
