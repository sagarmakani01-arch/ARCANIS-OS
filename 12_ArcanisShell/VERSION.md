# ArcanisShell

**Version:** 0.1.0  
**Status:** Alpha  
**Last Updated:** 2026-07-08

Initial implementation of the ArcanisShell next-generation shell.

## 0.1.0 — Foundation (2026-07-08)

- Shell engine with dual-mode execution (traditional + natural language)
- Command parser routing input to traditional commands or AI requests
- Traditional command set: file, process, system, and script execution
- AI interface: intent understanding, safe planning, suggestions, explanations, automation generation
- Security subsystem: permission policy, filesystem sandbox, append-only activity log
- Integration adapters for ArcanisBrain, ArcanisAgents, and ArcanisOS (local fallbacks)
- Unit test suite (19 tests) covering parser, commands, AI flow, and security
- Documentation: README, architecture, API reference, and examples
