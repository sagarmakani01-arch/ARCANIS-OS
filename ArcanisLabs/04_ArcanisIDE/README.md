# 04 — ArcanisIDE

AI-native development environment for the Arcanis ecosystem.

## Purpose

An IDE that understands code semantically, not just as text. Features include AI-assisted code completion, intelligent refactoring, integrated debugging, and a plugin system.

## Status

**Phase: Design** — Awaiting language and compiler APIs.
