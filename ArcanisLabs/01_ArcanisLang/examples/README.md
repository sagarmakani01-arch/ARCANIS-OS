# ArcanisLang Examples

| Example | Description |
|---------|-------------|
| `hello_world.arc` | Minimal program — print to stdout |
| `ai_assistant.arc` | AI agent with intent classification and tool use |
| `semantic_search.arc` | Document search using embeddings |
| `concurrent_agent.arc` | Parallel agent workflows with async/await |

All examples are written against the v0.1 language specification and will be validated once the compiler is functional.
