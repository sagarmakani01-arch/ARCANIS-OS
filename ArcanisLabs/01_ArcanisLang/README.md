# 01 — ArcanisLang

## Purpose

A beginner-friendly, AI-native programming language designed as the foundation of the Arcanis ecosystem. Features AI-native syntax, memory safety, and a path to self-hosting.

## Status

**Phase: Design** — Language specification v0.1 complete. Next: lexer implementation.
